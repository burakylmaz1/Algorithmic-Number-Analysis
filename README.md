# Algorithmic-Number-Analysis
A math-logic based Python application that detects Armstrong numbers by decomposing digits and calculating exponential sums.
## Öne Çıkan Özellikler
- **Basamak Ayrıştırma:** Modulo (`%`) ve tam sayı bölmesi (`//`) operatörleri ile dinamik ayrıştırma.
- **Üstel Hesaplama:** Sayı uzunluğuna (`len`) göre her basamağın kuvvetinin alınması.
- **Veri Koruma:** Geçici değişken (`gecici_sayı`) kullanımı ile orijinal veri üzerinde güvenli karşılaştırma.

## Teknik Detaylar
- **Dil:** Python 3.x
- **Yapılar:** `while` döngüleri, Tip Dönüşümü (Type Casting), Koşullu İfadeler.

## Nasıl Çalıştırılır?
1. Bilgisayarınızda Python yüklü olduğundan emin olun.
2. Dosyayı terminalden `python armstrong_checker.py` komutuyla çalıştırın.
